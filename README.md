Nekland Tools
=============

Just some classes helping to code with PHP in general.


Installation
------------

Inside your `composer.json` file:

```json
{
    "require": {
        "nekland/tools": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/Nekland/Tools"
        }
    ]
}
```

Then `composer update nekland/tools`.
